# 用户指南

## 快速开始

### 启动 Aemeath

```bash

```


## 常用功能

### 1. 工具调用

**查看日程：**

```
❯ 查看明天的日程
```

**发送邮件：**

```
❯ 发送邮件给 test@example.com，主题是"会议通知"，内容是"明天下午3点开会"
```

**搜索网页：**

```
❯ 搜索一下Python最新版本
```

### 2. 记忆系统

**添加记忆：**

```
❯ /remember 我的生日是1月15日
```

**查看记忆：**

```
❯ /memory
```

**搜索记忆：**

```
❯ /search 生日
```

### 3. 语音交互

**开启语音模式：**

```
❯ /voice on
```

**语音唤醒：**

说 "小爱" 即可唤醒助手。

**语音命令：**

- "停止" - 中断当前操作
- "暂停" - 暂停播放
- "继续" - 继续播放

### 4. 个性化

**切换人格：**

```
❯ /personality aemeath      # 爱弥斯模式（默认，基于鸣潮角色）
❯ /personality friendly    # 友好亲切
❯ /personality professional # 专业正式
❯ /personality casual      # 轻松随意
❯ /personality humorous    # 幽默风趣
❯ /personality tsundere    # 傲娇
❯ /personality genki       # 元气满满
```

**爱弥斯特有功能：**

- 会叫你"漂泊者"
- 使用星辉、共鸣等鸣潮世界词汇
- 说出角色经典台词
- 俏皮活泼的语言风格

**查看关系：**

```
❯ /relationship
```

### 5. 多模态

**分析图像：**

```
❯ /image screenshot.png
```

**解析PDF：**

```
❯ /pdf document.pdf
```

**截取屏幕：**

```
❯ /screenshot
```

## 配置文件

配置文件位于 `~/.aemeath/config.yaml`：

```yaml
model:
  provider: deepseek
  name: deepseek-v4-flash
  temperature: 0.7

voice:
  enabled: true
  voice_name: zh-CN-XiaoxiaoNeural
  wake_word: 小爱

memory:
  enabled: true
  max_short_term: 100

personality:
  type: friendly
  evolve: true
```

## 故障排除

### API密钥错误

确保 `.env` 文件中的密钥正确：

```bash
cat .env
```

### 语音不工作

检查麦克风权限：

```bash
# Linux
sudo usermod -aG audio $USER

# Mac
# 系统偏好设置 > 安全性与隐私 > 麦克风
```

### 记忆不保存

检查数据目录权限：

```bash
ls -la ~/.aemeath/data/
```

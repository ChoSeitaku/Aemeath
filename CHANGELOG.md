## 版本说明

### 版本号格式

`MAJOR.MINOR.PATCH`

- **MAJOR**: 不兼容的 API 变更
- **MINOR**: 向后兼容的功能添加
- **PATCH**: 向后兼容的 bug 修复

### 标签说明

- `新增` - 新功能
- `优化` - 功能优化
- `修复` - Bug 修复
- `废弃` - 即将移除的功能
- `移除` - 已移除的功能
- `安全` - 安全相关变更
